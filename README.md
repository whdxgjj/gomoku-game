# 五子棋游戏

一个简单而精美的网页版五子棋游戏，使用 HTML5 Canvas 和原生 JavaScript 开发。

## 功能特点

- 15x15 标准五子棋棋盘
- 黑白双方轮流下棋，黑棋先手
- 实时显示当前玩家和游戏状态
- 支持悔棋功能
- 自动判定胜负（先连成五子者获胜）
- 棋子采用渐变渲染，视觉效果精美
- 标记最后一步落子位置，方便查看
- 响应式设计，适配不同屏幕

## 快速开始

### 方式一：直接打开

直接在浏览器中打开 `index.html` 文件即可开始游戏。

### 方式二：使用本地服务器

```bash
# 安装依赖（如果需要）
npm install

# 启动本地服务器
npm run serve
```

然后在浏览器中访问 `http://localhost:8080`

## 游戏规则

1. 黑棋先手，双方轮流下棋
2. 点击棋盘交叉点落子
3. 先连成五子（横、竖、斜任意方向）者获胜
4. 可使用"悔棋"按钮撤销最后一步

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API
